# Анализ данных по продажам игр

## Описание проекта

Вы работаете в интернет-магазине **«Стримчик»**, который продаёт по всему миру компьютерные игры.  
Доступны исторические данные о продажах игр, оценках пользователей и критиков, жанрах и платформах (например, Xbox или PlayStation).  

Ваша задача — выявить закономерности, определяющие успешность игры.  
Это позволит делать ставки на потенциально популярный продукт и планировать рекламные кампании.  

Данные охватывают период до 2016 года. Представим, что сейчас декабрь 2016 года, и вы планируете кампанию на 2017-й.

**Примечание:**  
Аббревиатура **ESRB** (Entertainment Software Rating Board) — это ассоциация, определяющая возрастной рейтинг игр (например, «Для взрослых», «Для детей младшего возраста», «Для подростков»).

### Шаг 3. Проведите исследовательский анализ данных

1. Посмотрите, сколько игр выпускалось в разные годы.  
   - Важны ли данные за все периоды?  
2. Изучите, как менялись продажи по платформам:  
   - Выберите платформы с наибольшими суммарными продажами.  
   - Постройте распределение по годам.  
   - Определите характерный срок появления новых и исчезновения старых платформ.  
3. Выберите актуальный период для анализа, чтобы построить прогноз на 2017 год.  
4. Определите лидирующие платформы, их рост или падение. Выберите потенциально прибыльные платформы.  
5. Постройте график «ящик с усами» по глобальным продажам игр в разбивке по платформам и опишите результат.  
6. Исследуйте влияние отзывов пользователей и критиков на продажи внутри популярной платформы:  
   - Постройте диаграмму рассеяния.  
   - Посчитайте корреляцию между отзывами и продажами.  
   - Сравните с продажами на других платформах.  
7. Проанализируйте распределение игр по жанрам:  
   - Какие жанры самые прибыльные?  
   - Есть ли жанры с высокими и низкими продажами?

## Описание данных

| Столбец             | Описание |
|--------------------|---------------------------------------------------------|
| Name               | Название игры |
| Platform           | Платформа |
| Year_of_Release    | Год выпуска |
| Genre              | Жанр игры |
| NA_sales           | Продажи в Северной Америке (млн копий) |
| EU_sales           | Продажи в Европе (млн копий) |
| JP_sales           | Продажи в Японии (млн копий) |
| Other_sales        | Продажи в других странах (млн копий) |
| Critic_Score       | Оценка критиков (максимум 100) |
| User_Score         | Оценка пользователей (максимум 10) |
| Rating             | Рейтинг ESRB (возрастная категория) |

**Примечание:** Данные за 2016 год могут быть неполными.
