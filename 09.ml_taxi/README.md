# Прогнозирование заказов такси

## Описание проекта

Компания «Чётенькое такси» собрала исторические данные о заказах такси в аэропортах.  

Чтобы привлекать больше водителей в период пиковой нагрузки, нужно спрогнозировать количество заказов такси на следующий час.  

Постройте модель для такого предсказания.  

**Критерий качества:**  
- Метрика RMSE на тестовой выборке должна быть не больше 48.

## Инструкция по выполнению проекта

1. Загрузите данные и выполните их ресемплирование по одному часу.  
2. Проанализируйте данные.  
3. Обучите разные модели с различными гиперпараметрами.  
4. Сделайте тестовую выборку размером 10% от исходных данных.  
5. Проверьте данные на тестовой выборке и сделайте выводы.  

## Описание данных

Данные лежат в файле `/datasets/taxi.csv`. Скачать датасет.  

- **num_orders** — количество заказов такси (от англ. *number of orders*).
